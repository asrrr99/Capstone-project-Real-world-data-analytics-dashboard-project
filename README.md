# Child Mortality Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing child mortality data to uncover trends, high-risk regions, and key factors contributing to child deaths. The goal is to transform raw data into meaningful insights using data cleaning, analysis, and visualization techniques.

An interactive Power BI dashboard is developed to help users explore patterns and make data-driven decisions.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Analyze child mortality trends over time
- Identify top and lowest performing regions
- Discover important patterns in the data
- Provide actionable recommendations
- Build a professional and interactive dashboard

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel / CSV
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

- Real-world child mortality dataset
- Includes attributes such as:
  - Country / Region
  - Year / Month
  - Death Count / Mortality Rate
  - Age Group (Neonatal, Infant, Under-5)
  - Additional factors (if available)

---

## 🧹 Data Cleaning Steps

- Removed duplicate records
- Handled missing values
- Converted date columns into proper format
- Standardized column names
- Unpivoted data where necessary
- Ensured correct data types for analysis
